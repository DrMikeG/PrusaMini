# Summary

This is a simple display stand for your favorite wand. It has been sized to hold the Harry Potter style wands that you might get at Universal Studios. For those who have smaller print beds, like myself, there is a version of the stand that can be printed in 3 pieces.

Update:

- The 2-part wand holder has been simplified to allow for any of the House Crests to be glued over the seam.
- Crest for all four Houses has been added and sized for the wand holder.
